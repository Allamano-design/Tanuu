A high-end, **Glasspunk** inspired interactive birthday experience. This project features a transparent UI, parallax effects, and an interactive "gift unwrap" carousel to showcase personal memories.

## ✨ Features
* **Glassmorphism UI:** Layered glass panels with backdrop filters.
* **Interactive Carousel:** A custom-built slider for "unwrapping" memory cards.
* **Responsive Design:** Optimized for mobile and desktop viewing.
* **Performance:** Built with Vite for lightning-fast load times.

## 🚀 Tech Stack
* **Frontend:** React.js (Vite)
* **Styling:** CSS3 (Custom Glassmorphism) & Tailwind CSS
* **Deployment:** Vercel
